PHP Switch Script
================================================

A script for installing ppa:ondrej/php repository and switching between php versions 

## Installation

Clone the repo or download php-switch.sh

## Requirements

* Ubuntu
* bash

## Usage

  ./php-switch.sh add-repo
  
  ./php-switch.sh info
  
  ./php-switch.sh install 7.0
  
  ./php-switch.sh uninstall 7.0
  
  ./php-switch.sh ver
